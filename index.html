Introduction

SiVim is a branch of the popular variation of Vim, Neovim. It turns Neovim into an IDE, similarly to Lunar Vim, or NvChad. SiVim simplifies configuration for those who prefer a "ready out-of-the-box" experience.

Customizability

By default, SiVim ships with roughly twelve plugins that offer complete, but baseline functionality. Plugins can be added, or removed depending on the user through plugins.vim. This file exists under /.config/nvim/vim-plug, and each plugin is categorized for readability.

Future Plans

Sometime in the future, SiVim will be migrated fully to Lua. Unfortunately, some things are either unnecessarily difficult, or inefficient with just Vimscript. When this change will occur is unknown, but the re-write will come with many advancements, the main one being a more efficient dashboard.
